# 光匣

### 为胶片摄影师准备的随身曝光工具

出门拍胶片时，手机里不必再散落着测光表、计算器和备忘录。光匣把拍摄现场最常用的三件事放在一起：**测环境光、计算闪光功率、记住每台相机里装了什么胶卷。**

它完全离线，不需要账号，也不会上传你的相机和胶卷记录。

> 当前版本：**1.2.12** · 支持 Android 6.0 及以上
>
> [下载最新版 APK](https://github.com/nina-CP200/guangxia/releases/latest)

## 拍摄时能做什么

### 测光

把手机对准场景，像使用一只随身反射式测光表。

- **全局平均**适合街景、风景和光线均匀的画面。
- **中心点**读取画面中央约 15% 的区域，适合人物面部、亮部或暗部等明确对象。
- 可在**光圈优先**和**快门优先**之间切换，直接得到最接近三分之一档的推荐值。
- 胶卷 ISO 可以锁定，避免拍摄中误触。
- 支持曝光锁定和 ±3 EV 设备校准。

### 闪光

为手动闪光灯快速找到能落地的功率、光圈和距离组合。

- 输入闪光灯全功率 GN 和拍摄 ISO 后，调整光圈或距离，功率拨盘会自动跳到合适位置。
- 手动改变闪光功率时，光圈会跟随计算；距离保持不动。
- 功率覆盖 **1/1 到 1/256**，可选择 **0.1 EV** 或 **0.3 EV** 步长，适合 Godox AD200、iT30 Pro 等支持细分功率的闪光灯。
- 距离使用 ± 按钮，每次调整 0.1 米；也可以整体切换为英尺。
- GN 和拍摄 ISO 可以锁定，减少现场误操作。

计算以直闪、厂商标称全功率 GN 为基准。柔光箱、跳闪、滤色片和环境反射都会造成实际光损，重要拍摄建议配合试拍或闪光测光表确认。

### 胶卷备忘录

为每台相机建立一张卡片，打开应用就知道哪台机器里还有哪卷胶片。

- 记录相机名称、品牌和型号。
- 记录当前胶卷、ISO、装卷日期和备注。
- 卸卷后自动保留历史，不会丢失过去拍过的胶卷。
- 每台相机同一时间只保留一卷在机胶卷，避免重复装卷记录。

## 一次典型的拍摄流程

1. 出门前，在相机卡片里记下刚装入的胶卷和 ISO。
2. 到达现场，用全局平均观察整体光线，再用中心点检查主体。
3. 选定光圈或快门，让光匣给出另一项曝光参数。
4. 需要补光时，输入闪光灯 GN，按实际距离调整功率和光圈。
5. 拍完卸卷，在相机卡片中结束当前胶卷并保留历史。

## 关于测光精度

光匣读取手机摄像头的实际曝光参数来估算 EV100。不同手机的镜头、传感器和厂商自动曝光算法并不完全相同，因此它适合作为实用随身测光工具，但不宣称达到专业独立测光表的计量精度。

第一次使用时，建议在均匀光线下用灰卡与可信测光表对照，再通过“设备 EV 校准”保存补偿值。更换手机后应重新校准。

## 隐私与数据

- 不申请联网权限。
- 不需要注册或登录。
- 相机、胶卷和设置都只保存在设备本地。
- 删除相机前会确认；删除后，其胶卷历史也会一并移除。

## 安装

从 [Releases](https://github.com/nina-CP200/guangxia/releases/latest) 下载 `GuangXia-1.2.12.apk`，在 Android 手机上允许“安装未知应用”后打开安装即可。

1.2.12 是首个使用光匣正式密钥签名的版本。此前安装过 1.2.11 或更早 Debug 签名测试包的用户，需要先卸载旧版再安装；卸载应用会同时删除本地胶卷记录。后续正式版本可以直接覆盖升级。

Xiaomi 15 的取景预览性能仍在持续验证，进展见 [Issue #1](https://github.com/nina-CP200/guangxia/issues/1)。
